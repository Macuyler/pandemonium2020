# pandemonium2020

This is a flutter game based on the 2020 pandemic.

## Setup

1. [Install Flutter](https://flutter.dev/docs/get-started/install)
2. Download and Run:

```
git clone https://github.com/Macuyler/pandemonium2020.git
cd pandemonium2020
flutter run
```
